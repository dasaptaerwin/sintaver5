# SINTA versi 5.0

Repositori ini menyimpan berbagai pemikiran tentang SINTA, sebuah sistem pemantauan dan evaluasi karya dan dampak karya ilmiah, yang dikembangkan oleh Kemristekdikti.

Sampai hari ini telah ada beberapa artikel saya tentang SINTA. Nah sekarang saya punya banyak kawan baru yang telah menyumbangkan pemikiran. Secara informal, kami membuat sebuah tim, bernama Tim Sains Terbuka (TST). Tim intinya adalah: Surya Darma, Juneman Abraham, dan saya sendiri.
